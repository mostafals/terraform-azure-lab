# Terraform Azure Lab

Ce projet est un lab Terraform Azure destiné à apprendre et pratiquer les bases de Terraform
comme en environnement professionnel.

## Objectif
- Comprendre Terraform étape par étape
- Créer une infrastructure Azure simple
- Appliquer les bonnes pratiques
- Préparer le projet pour la CI/CD

## Infrastructure créée
- Resource Group
- Virtual Network (VNET)
- Subnet
